What is CSS?

  1. CSS Stand for Cascading Style Sheet

  2. CSS describe how HTML element are to be display on screen,paper,or in other media.

  3. Css is not necessary CSS saves a lot of work.It can control the layout of multiple web pages all at once

  4. External stylesheet are stored in CSS files

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elementsl

  • Internal - by using a style element in the head section

  • External - With an external style sheet, you can change the look of an entire website by changing just one file!

  • CSS comments are not displayed in the browser, but they can help document your source code.The external style sheet is generally used when you want to make changes on multiple pages. It is ideal for this condition because it facilitates you to change the look of the entire web site by changing just one file.

CSS selectors are used to select the content you want to style. Selectors are the part of CSS rule set. CSS selectors select HTML elements according to its id, class, type, attribute etc.

The id selector selects the id attribute of an HTML element to select a specific element. An id is always unique within the page so it is chosen to select a single, unique element.

The class selector selects HTML elements with a specific class attribute. It is used with a period character .
